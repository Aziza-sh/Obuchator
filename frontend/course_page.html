Создание адаптивного дизайна с CSS Grid и Flexbox

Время изучения: 45 минут

Введение в адаптивный дизайн

Адаптивный веб-дизайн — это подход к разработке веб-сайтов, который обеспечивает оптимальное отображение на различных устройствах и размерах экранов. Сегодня мы изучим две мощные технологии CSS: Grid и Flexbox.

Важно: Адаптивный дизайн стал стандартом с ростом использования мобильных устройств. Google учитывает адаптивность при ранжировании сайтов.

CSS Grid Layout

CSS Grid — это двумерная система разметки, которая позволяет создавать сложные макеты с строками и столбцами. Рассмотрим базовый пример:

/* Создание grid-контейнера */
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-gap: 20px;
}

/* Адаптивность для мобильных */
@media (max-width: 768px) {
  .container {
    grid-template-columns: 1fr;
  }
}

CSS Flexbox

Flexbox — это одномерная модель разметки, которая позволяет распределять пространство между элементами в контейнере. Идеально подходит для компонентов интерфейса:

/* Flexbox навигация */
.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
  flex-wrap: wrap;
}

.nav-item {
  flex: 1;
  min-width: 100px;
  text-align: center;
}

Комбинирование Grid и Flexbox

Для создания сложных адаптивных макетов часто используется комбинация Grid для общей структуры страницы и Flexbox для отдельных компонентов.

Преимущества такого подхода:

1. Grid идеально подходит для макета всей страницы

2. Flexbox отлично справляется с выравниванием элементов внутри ячеек

3. Упрощается создание адаптивных интерфейсов
