
# Обучатор

> Интерактивная образовательная платформа с интеграцией AI-технологий для работы с учебными материалами

**Обучатор** — это веб-приложение для изучения различных дисциплин с удобным интерфейсом и встроенными инструментами интеллектуальной обработки текста. Проект сочетает классический подход к обучению с современными возможностями машинного обучения.

---

## Особенности

-  **Курсы** — структурированные учебные материалы по разным предметам
-  **AI-обработка текста** — автоматическое выделение ключевых тезисов из учебных материалов на основе модели **ruT5**
-  **Регистрация и авторизация** — личный кабинет для отслеживания прогресса
-  **Тёмная тема** — комфортное использование в любое время суток
-  **Словарик** — интерактивный инструмент для изучения терминов по разделам
-  **Telegram-уведомления** — подписка на новости и обновления курсов

##  Технологический стек

### Backend
| Технология | Назначение |
|------------|------------|
| **Python + FastAPI** | API-сервер |
| **PostgreSQL** | Основная база данных |
| **Redis** | Кеширование и управление сессиями |
| **SQLAlchemy** | ORM для работы с БД |
| **Alembic** | Миграции базы данных |
| **JWT** | Аутентификация и авторизация |
| **python-telegram-bot** | Интеграция с Telegram |

### Frontend
| Технология | Назначение |
|------------|------------|
| **HTML5** | Структура страниц |
| **CSS3** | Стилизация и адаптивная вёрстка |
| **JavaScript** | Динамическое взаимодействие |
| **ITC Font** | Современная типографика |

### ML
| Технология | Назначение |
|------------|------------|
| **Python** | Бэкенд-логика |
| **PyTorch** | Работа с нейросетевыми моделями |
| **Transformers (Hugging Face)** | Загрузка и использование модели ruT5 |
| **scikit-learn** | Вспомогательные алгоритмы |
| **NLTK** | Обработка естественного языка |

### Модель AI
- **IlyaGusev/ruT5-base-sum-gazeta** — модель суммаризации текста, адаптированная для русского языка
- Поддерживается **GPU (CUDA)** для ускорения обработки



##  Установка и запуск

### 1. Клонирование репозитория
```bash
git clone https://github.com/Aziza-sh/Obuchator.git
cd Obuchator
```

### 2. Настройка backend

Перейдите в папку `backend` и создайте файл `.env` на основе примера:
```bash
cd backend
cp .env.example .env
```

Заполните `.env` своими данными:
```env
POSTGRES_DB=obuchator_db
POSTGRES_USER=user
POSTGRES_PASSWORD=password
POSTGRES_HOST=db
POSTGRES_PORT=5432
DATABASE_URL=postgresql+asyncpg://user:password@db:5432/obuchator_db
REDIS_URL=redis://redis:6379/0
SECRET_KEY=your_secret_key_here
ALGORITHM=HS256
ACCESS_TOKEN_EXPIRE_MINUTES=30
REFRESH_TOKEN_EXPIRE_HOURS=168
TELEGRAM_BOT_TOKEN=your_bot_token_here
SITE_URL=http://localhost:8000
```

Установите зависимости и запустите сервер:
```bash
pip install -r requirements.txt
uvicorn main:app --reload
```

### 3. Настройка ML-компонента

Перейдите в папку `ml` и установите зависимости:
```bash
cd ml
pip install -r requirements.txt
```

### 4. Запуск frontend-части

Откройте `frontend/index.html` в браузере или используйте любой локальный сервер:
```bash
# Например, с помощью Python
python -m http.server 8000
```
